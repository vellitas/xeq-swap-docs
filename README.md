## About

The Equilibria (XEQ) token swap allows holders of legacy XEQ to receive equivalent XEQ on the new Equilibria chain. Legacy XEQ is deposited to a published address monitored via a public view key. Once the swap window closes, the verified total is minted as a single genesis transaction on the new chain and distributed to recipients.

The legacy Equilibria chain will be permanently shut down shortly after the swap window closes. The Equilibria GitHub will be locked at that point — no further forks or client downloads will be possible.
